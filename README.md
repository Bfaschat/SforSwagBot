# SforSwagBot
### This is a chat bot on Telegram which helps to:-
#### 1.Get lyrics of songs
#### 2.Get restaurants and cafes by sending your location
#### 3.Knowing about the Menus of restaurants and cafes near you
#### 4.Get Random Quotes
